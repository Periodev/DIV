## 🎯 遊戲特色

- **13 個精心設計的關卡**
  - World 0：基礎機制教學 (6關)
  - World 1：時間線機制 (5關)
  - World 2：進階機制 (2關)

- **圖形界面關卡選擇器**
  - 方向鍵 ↑↓ 選擇關卡
  - ENTER 或 SPACE 進入關卡
  - ESC 退出

- **智能教學系統**
  - 首次進入自動顯示教學
  - 隨時按 H 查看教學
  - 進度自動保存

- **完整遊戲循環**
  - 選擇關卡 → 遊玩 → 完成 → 返回選擇器

## 🎮 操作說明

### 關卡選擇器
- **↑ ↓** - 選擇關卡
- **ENTER / SPACE** - 進入關卡
- **ESC** - 退出遊戲

### 遊戲中
- **W/A/S/D 或方向鍵** - 移動角色
- **X / SPACE** - 拾取/放下方塊
- **V** - 分裂/合併時間線
- **M** - 預覽合併
- **C** - 切換繼承模式
- **Tab** - 切換時間線視角
- **H** - 顯示/隱藏教學
- **F5 / R** - 重新開始
- **Z** - 撤銷
- **ESC** - 退出關卡

### 完成關卡後
- **SPACE** - 返回關卡選擇器

## 📊 進度保存

遊戲會在同目錄下自動創建 `progress.json` 文件保存進度：
- 記錄已完成的關卡
- 首次進入標記
- 可刪除此文件重置所有進度

## 🐛 故障排除

### 問題：雙擊後沒有反應
解決：
1. 檢查是否被防毒軟體阻擋
2. 以管理員身份運行
3. 檢查 Windows 事件查看器中的錯誤信息

### 問題：畫面顯示異常
解決：
1. 更新顯示卡驅動
2. 確保 Windows 已安裝最新更新
3. 嘗試以兼容模式運行

### 問題：進度丟失
解決：
- 確保 `progress.json` 文件與 exe 在同一目錄
- 不要將遊戲安裝在需要管理員權限的目錄
